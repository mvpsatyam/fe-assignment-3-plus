Repeating Linear Gradient

The repeating-linear-gradient function allows you to create a linear gradient background that repeats at regular intervals along a specified direction

  • Basic Syntax: The general syntax for repeating-linear-gradient is:

    repeating-linear-gradient([angle | to side-or-corner], color-stop1, color-stop2, ...);

    • [angle | to side-or-corner]: Specifies the direction of the gradient. It can be an angle (in degrees) or a keyword (to top, to right, to bottom, to left, to top right, etc.).

    • color-stop1, color-stop2, ...: Defines the colors and positions along the gradient line.

  • Understanding Color Stops: Each color stop defines a color and an optional position. The position is a percentage or a length along the gradient line where the color should be placed. If you omit positions, the colors will spread evenly.
